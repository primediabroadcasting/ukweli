Using the API

The API powering this is at http://ukweli.ngrok.io/api/

It accepts POST requests and expects a JSON payload with a single key called message that contains the message you want to test.

{
  "message": "please call us now to collect your prize!!"
}

In response you will get a JSON payload containing 3 keys:

  • known is the indicator on how closely your message matches our database of known fake messages
  • fakenews is how closely your message matches the indicators of fake news
  • smsspam is how closely your message matches the indicators of SMS spam

{
    "fakenews": 14.000000000000002,
    "known": 19,
    "smsspam": 73
}

Sample Code

Using curl

~ curl -H "Content-Type: application/json" 
  -X POST 
  -d '{"message":"Kindly note that the lifespan of a green barcoded  ID comes to an end 31/03/2018\n
  Please apply for your smart card ID\nLets share the information"}' http://ukweli.ngrok.io/api/
{
  "fakenews": 7.000000000000001,
  "known": 93.0,
  "smsspam": 21.0
}
